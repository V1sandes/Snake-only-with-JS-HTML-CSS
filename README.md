# Snake-only-with-JS-HTML-CSS

Классическая аркадная игра "Змейка", реализованная на HTML с CSS с использованием Java Script. Игрок управляет змейкой, которая собирает еду и растёт. Проект включает два режима: CLASSIC  и TELEPORT. Игра запускается в браузере и не требует установки.

## Стек технологий

**Frontend:** HTML5, CSS3, Vanilla JavaScript

## Деплой

Проект доступен по ссылке:
https://github.com/V1sandes/Snake-only-with-JS-HTML-CSS/issues

## Запуск локально

1. Склонируйте репозиторий

2. Перейдите в папку проекта

3. Откройте файл `index.html` в любом браузере.
